# AlgoSelectionMF



## Installation
```
git clone https://github.com/automl-private/AlgoSelectionMF/AlgoSelectionMF.git
cd AlgoSelectionMF
conda create -n mf_gravitas python=3.8
conda activate mf_gravitas

# Install for usage
pip install .

# Install for development
make install-dev
```
