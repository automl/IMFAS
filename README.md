# IMFAS



## Installation
```
git clone https://github.com/automl/IMFAS.git
cd IMFAS
conda create -n imfas python=3.8
conda activate imfas

# Install for usage
pip install .

# Install for development
make install-dev
```

Documentaiton at https://automl.github.io/IMFAS/main
